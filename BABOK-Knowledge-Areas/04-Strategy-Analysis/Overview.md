# Strategy Analysis
*IIBA BABOK Knowledge Area 4*

## Overview
Strategy Analysis knowledge area describes how business analysts identify business needs, assess current state, determine future state, and define change strategy.

## Key Tasks & Outputs

### Analyze Current State
- **Techniques**: SWOT Analysis, Business Capability Analysis
- **Outputs**: Current State Description
- **Templates**: [SWOT Analysis Template](../IIBA-Techniques/SWOT-Analysis-Template.md)

### Define Future State
- **Techniques**: Benchmarking, Goal Modeling
- **Outputs**: Future State Description
- **Artifacts**: Future State Models

### Assess Risks
- **Techniques**: Risk Analysis, Assumption and Constraint Analysis
- **Outputs**: Risk Assessment Results
- **Artifacts**: Risk Register

### Define Change Strategy
- **Techniques**: Gap Analysis, Feasibility Analysis
- **Outputs**: Change Strategy
- **Templates**: [Business Case Template](../IIBA-Templates-Artifacts/Business-Case-Template.md)

## Key Concepts
- **Current State**: Understanding the existing business situation
- **Future State**: Defining the desired business situation
- **Change Strategy**: Approach to move from current to future state

## Related Techniques
- SWOT Analysis
- Business Cases
- Risk Analysis
- Gap Analysis
- Benchmarking
